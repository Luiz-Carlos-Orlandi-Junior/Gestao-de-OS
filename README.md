Gestão de Ordens de Serviço - Back-End

Este repositório contém o desenvolvimento do back-end do projeto Gestão de Ordens de Serviço, uma aplicação voltada para auxiliar mecânicos autônomos e oficinas informais no gerenciamento de seus serviços e ordens de serviço de forma digital, segura e organizada.

O projeto tem como principal objetivo substituir processos realizados em papel por uma solução moderna, permitindo maior controle das informações, redução de perdas de dados e mais praticidade no dia a dia do profissional.

A aplicação busca proporcionar:

Gerenciamento de ordens de serviço;
Controle de clientes e veículos;
Organização de serviços realizados;
Armazenamento mais seguro das informações;
Maior credibilidade e profissionalismo no atendimento ao cliente;
Modernização dos processos utilizados por mecânicos informais e autônomos.

Este repositório é responsável exclusivamente pela camada back-end da aplicação, incluindo regras de negócio, APIs, autenticação, persistência de dados e integração com banco de dados.

Tecnologias utilizadas no projeto:

Node.js
NestJS
TypeScript
PostgreSQL

O front-end da aplicação será desenvolvido em um repositório separado.
